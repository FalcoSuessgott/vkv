Print the secrets recursively as JSON

### Required flags

```bash
vkv --path <path> --format=json
```

### Optional flags
| Flag                  | Description                                                                       | Env                    | Default |
|-----------------------|-----------------------------------------------------------------------------------|------------------------|---------|
| `--show-values`       | don't mask values                                                                 | `VKV_SHOW_VALUES`      | `false` |
