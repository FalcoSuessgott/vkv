Print the secrets recursively as YAML

### Required flags

```bash
vkv --path <path> --format=yaml
```

### Optional flags
| Flag                  | Description                                                                       | Env                    | Default |
|-----------------------|-----------------------------------------------------------------------------------|------------------------|---------|
| `--show-values`       | don't mask values                                                                 | `VKV_SHOW_VALUES`      | `false` |
